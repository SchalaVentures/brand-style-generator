# Support

## Getting Help

Thanks for using Brand Style Generator! Here's how to get help.

### Bug Reports

Found a bug? Please open an issue using the [bug report template](https://github.com/SchalaVentures/brand-style-generator/issues/new?template=bug_report.md). Include steps to reproduce, browser/OS info, and screenshots if applicable.

### Feature Requests

Have an idea? Open an issue using the [feature request template](https://github.com/SchalaVentures/brand-style-generator/issues/new?template=feature_request.md).

### Questions

For general questions about using Brand Style Generator, open a [GitHub Issue](https://github.com/SchalaVentures/brand-style-generator/issues) with the **question** label.

### Security Issues

**Do not open a public issue for security vulnerabilities.** See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## What This Repo Is Not

GitHub Issues are for bugs, feature requests, and questions about this project. They are not a general support channel for unrelated topics (e.g., help with your Tailwind config, Nhost setup, or deployment platform).
